touchy
======

Touch framework for web apps.

Here you could find the very first prototype of touchy framework (:

___
Primary usage instance:

    touch(document.body)
        .bind({
            start   : function( touches ){..},
            move    : function( touches ){..},
            end     : function( touches ){..},

            click   : function( touches ){..}
        });
